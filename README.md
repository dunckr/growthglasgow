# [GrowthGlasgow](http://www.growthglasgow.com)

Event page with RSVP.

#### Install

`npm i`

#### Run

`node .`
`gulp`

#### Env vars

* MONGO_URL
* TWITTER_CONSUMER_KEY
* TWITTER_CONSUMER_SECRET

### Build

`gulp production`

### Deploy

`git push heroku master`
